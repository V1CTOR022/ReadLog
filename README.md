📚 Sistema de Livraria (Projeto Acadêmico)
📌 Sobre o projeto

Este projeto foi desenvolvido como parte de um trabalho acadêmico com o objetivo de testar meus conhecimentos em banco de dados e explorar o uso do SQL na prática.

A aplicação consiste em um sistema simples de gerenciamento de livros, permitindo armazenar e organizar informações básicas sobre obras e seus autores.

🚀 Funcionalidades

➕ Cadastrar novos livros

✍️ Armazenar nome dos autores

📖 Registrar nome do livro

📅 Salvar data de cadastro ou publicação

📋 Visualizar lista de livros cadastrados

❌ Excluir registros

🛠️ Tecnologias utilizadas

SQL (SQLite) – armazenamento e manipulação dos dados

PHP – lógica do sistema e integração com o banco

HTML/CSS – interface do usuário

🎯 Objetivo

O principal objetivo deste projeto foi:

Praticar comandos SQL (SELECT, INSERT, UPDATE, DELETE)

Trabalhar com estrutura de tabelas e relacionamento simples

Desenvolver um CRUD básico

Consolidar conhecimentos em backend
