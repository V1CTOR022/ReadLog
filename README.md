📋 Cronograma de Tarefas (Projeto Acadêmico)
📌 Sobre o projeto

Este projeto foi desenvolvido como parte de um trabalho acadêmico com o objetivo de testar meus conhecimentos em banco de dados e explorar as possibilidades do SQL na prática.

A aplicação consiste em um sistema simples de gerenciamento de tarefas (cronograma), onde é possível adicionar, acompanhar e organizar atividades do dia a dia.

🚀 Funcionalidades

➕ Adicionar novas tarefas

📅 Definir data de término

✅ Marcar tarefas como concluídas

❌ Excluir tarefas

📊 Visualizar tarefas pendentes e concluídas

🛠️ Tecnologias utilizadas

SQL (SQLite) – armazenamento e manipulação dos dados

PHP – lógica do sistema e integração com o banco

HTML/CSS – interface bem simples do usuário

🎯 Objetivo

O principal objetivo deste projeto foi:

Praticar comandos SQL (SELECT, INSERT, UPDATE, DELETE)

Entender melhor a integração entre backend e banco de dados

Desenvolver um sistema funcional simples

▶️ Como executar o projeto

Instale um ambiente local (ex: XAMPP)

Coloque os arquivos na pasta htdocs

Inicie o servidor Apache

Execute o projeto no navegador:

http://localhost/cronograma
📚 Aprendizados

Durante o desenvolvimento, aprendi:

Estruturação de banco de dados simples

Manipulação de dados com SQL

Organização de código em PHP

Criação de um CRUD completo
